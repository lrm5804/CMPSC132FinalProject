
**DESCRIPTION:**
This NumberGuesser game is a terminal-based Python game. Users are repeatedly prompted to input an integer between 0 and 100. They get to set their level of difficulty to easy, medium, or hard. This determines how many attempts they are allowed before they win or lose the game. The Easy mode allows 20 attempts, the Medium allows 10 attempts, and the Hard mode allows 5 attempts. Users are prompted until they either guess the number and win or run out of attempts and lose. Either way, the program guides them through with messages throughout. With each valid guess, they receive feedback for whether their guess was too low or too high. Users can play as many times as they'd like and the program will keep track of their wins.

**HOW TO RUN PROGRAM:**
1. Open VS Code and run NumberGuesser.py file.
2. Type 'python3 -i NumberGuesser.py' if you're running on Mac or 'python -i NumberGuesser.py' if you're running on Windows. Hit enter.
3. Type 'x = NumberGuesser()'. Hit enter.
4. Type 'x.begin()'. Hit enter.
5. Play the game!
